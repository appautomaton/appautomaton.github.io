Quickstart

Three commands.

Clone, symlink, point your agent at it. Works the same under Claude Code, Codex, and Gemini.

terminal
~/skills
$ git clone https://github.com/appautomaton/document-SKILLs
 
$ cd document-SKILLs
$ for s in docx pdf pptx xlsx; do
ln -s "$(pwd)/$s" ~/.claude/skills/$s
done
 
$ claude "build a PDF report from sales.csv"
✓ skill `pdf` invoked
